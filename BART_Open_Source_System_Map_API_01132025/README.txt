BART App Map

An open source system map for the San Francisco Bay Area Rapid Transit District (BART) system.


##BART System Map

An open source BART system map designed for developers who want to include a simple BART map in their applications.


##Author

San Francisco Bay Area Rapid Transit District (BART), adapted from the original work by Trucy Phan (trucy@blinktag.com) in 2014.

The map has been modified by BART Staff and vendors, and is current as of September 11, 2024.

##Source

The 'app map' is available on BART's website (http://www.bart.gov/schedules/developers/maps.aspx) in .ai and .png formats.


##Example

Use the .ai file to change the base map, line colors and widths, fonts, etc.
